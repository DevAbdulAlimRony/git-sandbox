- A README file is used to communicate important information about a repository including: what the project dos, how to run the project, why it's noteworthy, who maintains the project.

- READMEs are markdown files, ending with the .md extension. Markdown is a convenient syntax to generate formatted text. Text to HTML conversion tool.

- Github Gists are a simple way to share code snippets and useful fragments with others. Gists are much easier to create, but offer far fewer features than a typical Github repository.

-  Github Pages: Host your web page. Pure HTML, CSS, Javascript
